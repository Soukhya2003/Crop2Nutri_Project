# Crop2Nutri – Crop to Nutrition to Patients

## Minor Project – MSc Data Science

### Project Overview
Crop2Nutri is a conceptual AI-integrated platform designed to reduce agricultural food waste by connecting surplus crops from farmers to hospitals and NGOs based on nutritional requirements.

### Objectives
- Reduce crop wastage
- Enable nutrient-aware matching
- Improve food redistribution efficiency
- Support healthcare institutions with targeted nutrition

### Technologies (Conceptual Design)
- Python (FastAPI – ML Microservice)
- TensorFlow (EfficientNetB0 for Crop Identification)
- PostgreSQL & Supabase (Database)
- Next.js (Frontend Architecture)
